# 📋 Responsive Semantic Survey Form

A clean, semantic HTML5 survey form designed to capture user feedback. This project demonstrates core web development skills, including structural layout, diverse input handling, data validation, and form accessibility.

## 🚀 Features
* **Semantic Structure**: Built using native HTML5 semantic tags for optimized accessibility.
* **Form Validation**: Features required fields, strict email formatting, and numeric boundaries (`min`/`max`).
* **Diverse UI Inputs**: Implements standard interactive web controls including text fields, dropdown menus, radio options, checkboxes, and text areas.

## 🛠️ How to Run
1. Download the `index.html` file.
2. Double-click the file to open and interact with the form instantly in any web browser (Chrome, Safari, Edge, etc.).
